Oboetoki는 이미지에서 단어를 추출하여 나만의 단어장을 손쉽게 관리할 수 있는 스마트 단어장 안드로이드 어플리케이션입니다.


## ✨ 주요 기능

* **스마트 OCR 인식:** 카메라나 갤러리 이미지에서 텍스트를 인식하여 빠르고 간편하게 단어를 추출합니다.
* **커스텀 단어장:** 추출한 단어 또는 직접 입력한 단어를 저장하고 관리할 수 있습니다.
* **단어 필터링 및 정렬:** 암기 완료 / 미암기 단어를 필터링하고 최신순, 오래된순으로 쉽게 정렬하여 학습 효율을 높입니다.
* **다국어 지원 (영어, 일본어):** 입력 시 학습할 언어를 직접 선택하여 구분할 수 있습니다.

## 🛠 기술 스택

* **Language:** Java, XML
* **Minimum SDK:** 24 (Android 7.0)
* **Target SDK:** 34 (Android 14)
* **Architecture / UI:** ViewBinding, AndroidX
* **Networking / Async:** Retrofit2, OkHttp, Gson, Coroutines
* **Security:** EncryptedSharedPreferences


## 🚀 시작하기

### 시스템 요구사항 (Prerequisites)
* **IDE:** Android Studio (최신 버전 권장)
* **Java:** JDK 11 이상
* **OS:** Android 7.0 (API 24) 이상 호환 기기 (Target: Android 14, API 34)
