# 목적
- 앱 UI 디자인 시안 구현

# 앱 정보
- 안드로이드 앱
- 이름 : Oboetoki 
- 앱 로고 (토끼 머리 실루엣)
- 주제 : 외국어 단어장 어플

# 제약조건
- android studio에서 실행 및 테스트할 계획입니다.  
- vscode, antigravity에서 실행하기 위한 코드, 의존성을 주입하지 마시오.  
- 백엔드 api는 추후에 적용할 수 있도록 경로만 지정해두시오.   
- 로고, 아이콘은 더미 리소스를 사용하시오. (추후 이미지 리소스로 교체 예정.)  
- minimum sdk: API 24 ("Nougat"; Android 7.0)
- build configuration language: Kiotlin DSL (build.gradle.kts)

# 주요 기능
- 로그인
    - 아이디
    - 비밀번호
- 회원가입
    - 아이디 (중복확인 버튼 존재)
    - 비밀번호
    - 이름
    - 전화번호 
    - 이메일 
- 실물 단어장 사진을 찍으면, 해당 페이지의 단어가 앱 단어장에 등록됨
- 단어장 관리 
    - 등록된 단어 리스트 확인 및 수정
    - 각 단어 원어 및 한국어 뜻 표시
    - 각 단어 클릭 시 단어 상세 정보 표시 및 수정 가능
        - 원어
        - 한국어 뜻
        - 등록 날짜
        - 기억 여부
        - 기억 정도

# 페이지 (주어진 html을 그대로 구현하시오)
## 스플래시 스크린 (splash_screen.html)
- 로고 및 앱 이름을 중앙에 표시
- 앱 실행 시 2초 정도 표시 후 Fade out으로 사라짐

## 로그인 페이지 (signin.html)
- 인덱스 페이지 이후, 사용자가 미 로그인 상태 시 표시
- 좌측 상단에 로고 및 앱 이름 작게 표시
- 아이디, 비밀번호 입력창
- 자동 로그인 체크 가능
- 하단에 로그인 버튼 배치
- 로그인 버튼 하단에 회원가입 버튼 배치

## 회원가입 페이지 (signup.html)
- 아이디 입력창 및 중복 확인 버튼
- 비밀번호 및 비밀번호 확인 입력창
- 이름 입력창
- 전화번호 입력창
- 이메일 입력창
- 회원가입 버튼
- 취소 버튼

## 메인 페이지 (main.html)
- 대시보드 느낌
- 상단에 연속 출석 일수 표시
- 중간에 등록된 단어 수, 외운 단어 수 및 퍼센트 표시
- 하단에 네비게이션 바 배치 (홈, 카메라, 단어장)

## 단어장 페이지 (vocabulary_list.html)
- 상단에 검색창 배치
- 검색창 좌측에 필터 버튼 배치
- 필터 버튼 클릭 시 모달창 표시 (vocabulary_filter_modal.html)
    - 아직 못 외운 단어
    - 외운 단어
    - 최근 등록 순
    - 기억 정도 낮은 순
- 단어를 표 형태로 표시
    - 원어를 좌측에, 한국어 뜻을 우측에 배치
    - 단어 클릭 시 모달창 표시 (vocabulary_detail.html)
        - 원어 (클릭 시 편집 가능)
        - 한국어 뜻 (클릭 시 편집 가능)
        - 등록 날짜 
        - 기억 여부(완료 or 미완료)
        - 기억 정도(%)

## 카메라로 단어장 촬영 후 OCR로 추출된 단어 리스트 표시 페이지
- vocabulary_list.html 파일을 참고하여 단어장 페이지와 유사한 디자인으로 구성
- 검색, 필터 등의 기능은 제외하고, 단어 리스트만 표시
- 각 단어 및 한국어 뜻 터치 시 해당 문자를 수정할 수 있도록 하시오.
- 하단에 추가, 취소 버튼을 배치하시오.
- 하단 네비게이션 바는 표시하지 마시오.
